Hakkımızda

StartupStore, modern e-ticaret çözümleri sunan bir startup demo projesidir. Bu uygulama, Progressive Web App (PWA) yaklaşımıyla geliştirilmiştir.

Kullanılan Teknolojiler
  • API: DummyJSON (https://dummyjson.com)
  • CSS Framework: Bootstrap 5
  • PWA: Service Worker + Manifest
Offline Çalışma

Uygulama, servis worker kullanarak temel dosyaları önbelleğe alır. İnternet bağlantısı olmadığında offline fallback sayfası gösterilir.
